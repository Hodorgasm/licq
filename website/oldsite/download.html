http://www.sjaensch.org/licq1.2.7-qt3.3-sprintf.patch QT 3.3 special-characters-bug workaround
As an increasing number of people seem to be experiencing a problem with locale-specific characters (e.q. accented characters, german umlauts and the like) are not beeing displayed correctly, Stephan Jaensch contributed a patch for this problem which workarounds the QT 3.3 problem with Licq 1.2.7.
A fix for this problem is already in Licq's CVS tree, this patch is just for convenience for those users who are not brave enough to use CVS ;-)
