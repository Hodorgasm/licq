Help

There are many ways to get help if you have any troubles with Licq:

  • Readme file
  • The Licq FAQ
  • Hints for using Licq.
  • Licq Install Guide for Slackware (very outdated!)
  • README.* - there are a number of documents included in the root directory both the main Licq program and the Qt-GUI plugin dealing with certain specific problems.
  • The Licq Mailing List and archive.
  • #Licq on irc.freenode.net
  • As a last resort email me.
