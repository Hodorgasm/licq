Bugs

When submitting bugs please make sure do the following:

  1. Check the open and closed bugs at the Bug Tracking site.
  2. Check the Licq mailing lists.
  3. Ensure that the problem can be reproduced with the latest CVS Snapshot. If it cannot be reproduced then please do not report it.
  4. Ensure that you have included an email address where you can be reached for questions.
  5. Be as descriptive as you can about the bug. Include OS, OS Version, QT Version if you are using the qt-gui, Error messages, etc.. Not just "It does not work".
  6. Regularly check your own Bug Reports as notes will be added by one of the developers.
If you have found a new bug, please submit it to Bug Tracking or subscribe to the LICQ Developers Mailing List and post your Bug Report there.

Remember in order for the developers to resolve bugs they need to communicate with the people reporting them. Please keep in touch with them and answer any questions that they may have to the best of your ability.
