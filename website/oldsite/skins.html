Skins for Licq work with the Qt-GUI plugin and are easy to make (see the SKINS.HOWTO) and easy to install. To use one, simply unpack it into the qt-gui directory (/usr/local/share/licq/qt-gui or /usr/share/licq/qt-gui) and then select it from the Skin Browser, or use the -s option to the qt-gui. You can also test a skin by unpacking it anywhere and passing the gui the full pathname to the directory.

There are also Icon Packs available to change the status icons in the contact list. Simply unpack them into the qt-gui directory as well.

See the official Licq Skins/Icons homepage for more skins, icons and where to submit new ones.
